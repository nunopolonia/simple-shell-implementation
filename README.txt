Passos para compilação:
  Existe um makefile associado ao projecto, logo apenas é necessário correr o comando
  
    make

  para a correcta compilação do projecto.

  Também podem ser invocado o comando:
  
    make clean
  
  ou o comando: 
  
    make rebuild
  
  que invocam diferentes targets no makefile.

Passos para execução
  O executavel gerado tem o nome sosh sendo portanto necessário correr o comando

    ./sosh

  para o executar.
